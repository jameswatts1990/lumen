## OpenDyslexic local assets

Place the OpenDyslexic binaries in this folder using the filenames below:

- `OpenDyslexic-Regular.woff2`
- `OpenDyslexic-Regular.woff`
- `OpenDyslexic-Regular.otf`
- `OpenDyslexic-Bold.woff2`
- `OpenDyslexic-Bold.woff`
- `OpenDyslexic-Bold.otf`

`index.html` is configured to resolve local files first (with `?v=2026-03-31` cache-busting query params), then fall back to CDN-hosted OTF files.
